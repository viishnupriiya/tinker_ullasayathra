# Ullasayathra - The Serendipity Travel Engine

A travel planning app that focuses on the "vibe" rather than just the destination. Built with React and Vite.

## Features
- **Vibe Check**: A fun quiz to find your travel persona.
- **Smart Search**: Find destinations like Kashmir, Kerala, Goa, etc.
- **Dynamic Itinerary**: Generates a day-by-day plan based on how many days you want to travel.
- **Mystery Mode**: "The Ottamind" lets you unlock your trip one day at a time.
- **Social Map**: See where your friends are traveling (mock data).

## How to Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm run dev
   ```
3. Open the link shown in the terminal (usually `http://localhost:5173`).

## Technologies Used
- React
- CSS (Custom styling)
- Leaflet (Maps)
